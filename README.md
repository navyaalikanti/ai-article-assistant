# AI Article Summarizer Chrome Extension

An AI-powered Chrome extension built using the Gemini API that helps users quickly understand web content by generating concise summaries, analyzing multiple tabs at once, and answering contextual questions directly from the article. The extension is designed to improve productivity by integrating Generative AI seamlessly into the browsing experience.

## Features

✔ Article summarization  
Summarizes the currently opened web article in multiple formats:
- Brief summary
- Detailed summary
- Bullet-point summary

✔ Multi-tab summarization  
Reads and analyzes content from all open tabs and generates a combined summary.

✔ Contextual question-answering  
Allows users to ask questions related to the article content and receive accurate, context-aware answers.

✔ Real-time content extraction  
Automatically extracts readable text from web pages when the extension is opened.

✔ Productivity-focused design  
Minimizes reading time and helps users grasp key points faster.

## How It Works

1. Open any web article in the browser  
2. Click the Chrome extension  
3. Choose a summary format (brief, detailed, or bullet points)  
4. Use multi-tab mode to summarize all open tabs  
5. Ask questions related to the article using the chat interface

## Tech Stack

- JavaScript  
- Chrome Extensions API  
- Gemini API (Generative AI)  
- HTML, CSS
